Important: instalation of Smarty - top

In your local server you must create folder smarty in your www folder and extract all files there. If you upload website to your domain (ex:www.example.com) you must find table website_fronted in your sql db and left row website_frontend_directory blank to work properly.

A) Configuration of MYSQL database and PHP code - top

How to install your MYSQL database:

  1. Create new database in PHPMYADMIN at your host. Call it 'smarty'.
  2. In the MYSQL folder there is sql.sql file that should be imported to your database 'smarty'.
  3. In the includes folder there is connection.inc.php file. You must configure there your database options, name, host, username and password which should be complement with data in phpmyadmin. Connection.inc file should be like in the image bellow.

Word about PHP code:

PHP code is widely used in script, especially in connecting with MYSQL database. Code is commented, and properly sorted, and can be easily understand. Also security is special matter. So PDO statement is used to prevent hackers attacks, Mysql injections, and there is also xss-security.php in security folder to prevent xss script attacks.
You can add, edit and delete content and website options in your admin panel so there isn't so much need to change php codes inside script, but if have php knowledge you can do it by yourself.

B) HTML Structure - top

This theme is a responsive layout with two columns. It' divided into 4 part. Header with id="header", horizontal navigation (which is located in includes/navigation.php file for desktop and includes/mobile-navigation.php for mobile devices).
Third part is content area (id="content"), which consist from breadcrumbs (id="crumbs"), left column (id="two-third-column") and right, smaller column (id="one-third-column"). Left, main column is place where is located your main content, last products reviews (id="short-review-box"), blogs, contact and privacy information depending of page.
Footer is fourth part of website and html code is located in includes/footer.php

If you would like to edit the css, font, or style of any elements in one of these columns, you would change it in css/main.css.
Admin part of script is located in admin folder and there are main files where php code for admin features is located. Admin css can be change in css folder of the website.
In admin folder is also located CKE editor that is used by our script for textarea of admin panel.

C) CSS Files and Structure - top

In css folder there are 3 main css files:

  • main.css
  • admin.css
  • reset.css

There are also 2 css folders, color_css, and font_css where font and color styles of the script are located. You can change it, edit it and add new styles here.

D) Admin structure - top

Admin area of script is most important area. Here you can control content of your website by creating, editing and deleting products, categories and brands as well as blog articles. You can also control basic website options.
There is an info icon to the right side of every input at every admin page where there is short explanation of input purpose. Read it carefully and avoid any misunderstading.

Main links in admin area are:

  • Site options
  • Navigation links
  • Products
  • Categories
  • Brands
  • Blog articles
  • Logout
  • Website
